  PASO 1: PRERREQUISITOS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

- Node.js 20+ instalado
- npm 10+
- Cuenta OpenAI con acceso a GPT-4o
- Cuenta Paddle (paddle.com) — gratuita para empezar
- Cuenta Vercel o Railway para deploy (opcional)

  PASO 2: CONFIGURAR AUDIOS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Los archivos de audio van en: frontend/public/audio/

  background.mp3  → Música de fondo (ya incluida)
  voice.mp3       → Archivo de voz (ya incluida)

IMPORTANTE — Timing de la voz:
Tu archivo voz.mp3 contiene 2 frases.
Debes detectar en qué segundo comienza la segunda frase:
1. Abrí el archivo en cualquier editor de audio (Audacity, gratuito)
2. Buscá el inicio de "Y no quiero volver a verte mal vestida"
3. Anotá el timestamp (ej: 5.2 segundos)
4. En frontend/.env.local: NEXT_PUBLIC_VOICE_DOWNLOAD_START=5.2

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  PASO 3: CONFIGURAR PADDLE (PAGOS)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Creá cuenta en paddle.com
2. En el dashboard → Catalog → Products
   - Crear producto: "Miroir Analysis"
   - Precio: USD 1.50
   - Tipo: one-time
   - Copiar el Price ID (ej: pri_01abc...)

3. En Developer → Authentication
   - Copiar Client-side token (para frontend)

4. En Developer → Notifications
   - Crear webhook: https://tu-backend.com/webhooks/paddle
   - Eventos: transaction.completed
   - Copiar Webhook secret

5. Para Argentina: Paddle opera como "merchant of record"
   y acepta Apple Pay y Google Pay globalmente.
   No necesitás cuenta bancaria Argentina.

  PASO 6: DEPLOY EN PRODUCCIÓN
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

OPCIÓN A: Vercel (Frontend) + Railway (Backend) — RECOMENDADO

Frontend → Vercel:
  1. vercel.com → New Project → importar miroir/frontend
  2. En Environment Variables: pegar valores del .env.local
  3. Deploy automático

Backend → Railway:
  1. railway.app → New Project → Deploy from GitHub → miroir/backend
  2. En Variables: pegar valores del .env
  3. Railway genera URL automática
  4. Copiar esa URL en NEXT_PUBLIC_API_URL del frontend

OPCIÓN B: VPS (DigitalOcean, Linode, etc.)
  1. Instalar Node 20, PM2, Nginx
  2. Clonar proyecto
  3. npm install && npm run build en ambas carpetas
  4. PM2: pm2 start dist/index.js --name miroir-api
  5. Nginx como reverse proxy

  PASO 7: DOMINIO PERSONALIZADO
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Comprar dominio (ej: miroir.app)
2. En Vercel → Domains → agregar tu dominio
3. En Railway → Settings → Domain → agregar subdominio (api.miroir.app)
4. Actualizar NEXT_PUBLIC_API_URL=https://api.miroir.app
5. Actualizar FRONTEND_URL=https://miroir.app en el backend
